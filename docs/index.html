File Reader

File reader is designed to access file on the internet via QR code. And it works with File Sharer.

Usage to share your file: (1)Download the APP from store and Open it.
(2)Press "Link To Dropbox" which may request permission to access your dropbox account.
(3)Import any file you wants to share to APP.
(4)Open the file.
(5)Slide to left and the QR Code will appear.

Usage to download file via QR Code:
(1)Download the APP from store and Open it.
(2)Press the “Add” button on the top right menu bar. APP would ask for permission of camera.
(3)Let the QR Code below on the center of the camera. The camera will dismiss if camera scans QR Code.
(4)Wait for download process.

How to create QR code?

Link to your dropbox account inside the APP.
